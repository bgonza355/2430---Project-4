# Programming Project 4: Monopoly Movement Simulation

## 1. Project Overview
This project is a computational model that simulates player movement in the classic board game Monopoly (2018 U.S. Rules). The goal is to calculate the empirical landing probability for each of the 40 squares on the board to determine which locations are visited most frequently across large datasets.

## 3. Implemented Strategies
The program compares two distinct approaches to leaving Jail to see how they affect long-run landing probabilities:

Strategy A (Immediate Exit): Use a "Get Out of Jail Free" card immediately or pay $50 to leave on the next turn.

Strategy B (Try for Doubles): Attempt to roll doubles for up to three turns. If unsuccessful, pay $50 and leave on the fourth turn.
