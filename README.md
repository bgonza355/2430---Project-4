Programming Project 4: Monopoly Movement Simulation1. Project OverviewThis project is a computational model designed to simulate player movement in the classic board game Monopoly (2018 U.S. Rules). The goal is to calculate the empirical landing probability for each of the 40 squares on the board.The simulation focuses specifically on movement mechanics, including:Standard dice rolls and movement."Three doubles in a row" jail rules.Chance and Community Chest card effects (movement and Jail).Comparison of two distinct "Get Out of Jail" strategies.2. Implemented Jail StrategiesThe simulation evaluates two different approaches to leaving Jail:Strategy A (Immediate Exit): The player uses a "Get Out of Jail Free" card immediately or pays $50 to leave on the next turn.Strategy B (Try for Doubles): The player attempts to roll doubles for up to three turns. If unsuccessful, they pay the $50 and leave on the fourth turn.4. How to Run the ProgramThis project is written in Java and requires no external libraries.CompilationNavigate to the root of the source directory and compile the files:Bashjavac Src/*.java
ExecutionRun the main simulation class:Bashjava Src.MonopolySimulation
The program will output data for $n = 1,000$, $10,000$, $100,000$, and $1,000,000$ turns across 10 independent runs for both strategies.5.
